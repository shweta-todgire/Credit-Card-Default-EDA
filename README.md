# Credit Card Default EDA

## Overview
This project performs Exploratory Data Analysis (EDA) on the Default of Credit Card Clients dataset to understand customer behavior and identify factors associated with credit card default.

## Dataset
- **Dataset:** Default of Credit Card Clients
- **Rows:** 30,000
- **Columns:** 25

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
- Data Inspection
- Descriptive Statistics
- Outlier Detection
- Histogram
- Box Plot
- Scatter Plot
- Bar Chart
- Count Plot
- Correlation Heatmap
- Outliers

## Key Findings
- Most customers did not default on their payments.
- Payment delay history (PAY_0–PAY_6) has the strongest relationship with default.
- Age and demographic features have a weak impact on default.
- Only 167 outliers were found, indicating good data quality.

## Conclusion
The analysis shows that payment history is the most important factor in predicting credit card default. These insights can help financial institutions identify high-risk customers and improve credit risk management.
